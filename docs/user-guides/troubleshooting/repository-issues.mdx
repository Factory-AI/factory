## Common Repository Issues

### Size Limit Exceeded

<Accordion title="Repository Too Large for Local Indexing">
  **Problem**: Your repository exceeds the 5M token limit for local indexing.

  **Solution**:

  1. Use remote repository integration:

     * Connect via GitHub or GitLab

     * Access full context features

     * Enable automatic updates

  2. Continue with local file sync:

     * Real-time file updates

     * Active development support

  **Learn More**: Visit our [Local File Sync](/user-guides/managing-context/local-file-sync) guide
</Accordion>

### Sync Issues

<Accordion title="Local File Sync Not Working">
  **Common Causes**:

  * Missing `.git` directory

  * Insufficient permissions

  * Network connectivity issues

  **Solutions**:

  1. Verify repository structure:

     ```bash
     ls -la .git  # Should show git directory contents
     ```

  2. Check permissions:

     ```bash
     ls -l .  # Should show write permissions
     ```

  3. Test network connection:

     ```bash
     ping app.factory.ai
     ```
</Accordion>

### Integration Problems

<AccordionGroup>
  <Accordion title="GitHub Integration Issues">
    **Common Issues**:

    * OAuth token expiration

    * Repository access permissions

    * Organization access settings

    **Solutions**:

    1. Verify OAuth status in settings

    2. Check repository permissions

    3. Review organization settings
  </Accordion>

  <Accordion title="GitLab Integration Issues">
    **Common Issues**:

    * API access token expiration

    * Instance connectivity

    * Project visibility settings

    **Solutions**:

    1. Regenerate API token

    2. Verify instance accessibility

    3. Check project permissions
  </Accordion>
</AccordionGroup>

## Diagnostic Tools

Use these commands to diagnose common issues:

```bash
# Check repository size
git count-objects -vH

# Verify remote connections
git remote -v

# Test API access
curl -I https://api.github.com/user
```
