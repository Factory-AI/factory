Connect to Google's Gemini models for advanced AI capabilities with multimodal support.

## Configuration

Add these configurations to `~/.factory/config.json`:

```json
{
  "custom_models": [
    {
      "model_display_name": "Gemini 2.5 Pro [Google]",
      "model": "gemini-2.5-pro",
      "base_url": "https://generativelanguage.googleapis.com/v1beta/",
      "api_key": "YOUR_GEMINI_API_KEY",
      "provider": "generic-chat-completion-api",
      "max_tokens": 32000
    },
    {
      "model_display_name": "Gemini 1.5 Pro [Google]",
      "model": "gemini-1.5-pro",
      "base_url": "https://generativelanguage.googleapis.com/v1beta/",
      "api_key": "YOUR_GEMINI_API_KEY",
      "provider": "generic-chat-completion-api",
      "max_tokens": 1048576
    },
    {
      "model_display_name": "Gemini 1.5 Flash [Google]",
      "model": "gemini-1.5-flash",
      "base_url": "https://generativelanguage.googleapis.com/v1beta/",
      "api_key": "YOUR_GEMINI_API_KEY",
      "provider": "generic-chat-completion-api",
      "max_tokens": 1048576
    }
  ]
}
```

## Getting Started

1. Go to [makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
2. Click "Create API Key"
3. Copy your API key
4. Add it to your Factory configuration

## Available Models

### Gemini 2.5 Pro
- **Context**: 32K tokens
- **Strengths**: Latest model with improved reasoning
- **Use Cases**: Complex tasks, code generation, analysis

### Gemini 1.5 Pro
- **Context**: Up to 1M tokens
- **Strengths**: Extremely long context, multimodal
- **Use Cases**: Document analysis, large codebases

### Gemini 1.5 Flash
- **Context**: Up to 1M tokens
- **Strengths**: Faster inference, cost-effective
- **Use Cases**: High-volume tasks, quick responses

## Features

- **Long Context**: Up to 1M tokens for Pro and Flash models
- **Multimodal**: Support for text, images, and other modalities
- **Fast Inference**: Optimized for low latency
- **Cost Effective**: Competitive pricing with free tier

## Pricing

Google offers a generous free tier:
- **Free Quota**: 60 requests per minute
- **Daily Limits**: Varies by model
- **Paid Tier**: Pay-as-you-go for higher limits

Check [ai.google.dev/pricing](https://ai.google.dev/pricing) for current rates.

## Regional Availability

Gemini API availability varies by region. Check [ai.google.dev/available_regions](https://ai.google.dev/available_regions) for supported locations.

## Advanced Features

### Safety Settings
Gemini models include built-in safety filters that can be configured through the API.

### System Instructions
Gemini supports system instructions for consistent behavior across conversations.

## Notes

- Base URL uses the `v1beta` API version
- Gemini models use the `generic-chat-completion-api` provider type
- Some features like function calling may have limited support
- Rate limits apply to free tier usage
