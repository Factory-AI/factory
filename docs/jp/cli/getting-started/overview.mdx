## 30秒で始める

<CodeGroup>```bash macOS/Linux
curl -fsSL https://app.factory.ai/cli | sh
```

```powershell Windows
irm https://app.factory.ai/cli/windows | iex
```

```bash npm
npm install -g droid
```</CodeGroup>

その後、プロジェクトに移動してdroid CLIを開始します。

<CodeGroup>```bash
# Navigate to your project
cd /path/to/your/project

# Start interactive session
droid
```</CodeGroup>

これでターミナルからFactory の開発エージェントに接続されました。[5分間のクイックスタートを試す →](/jp/cli/getting-started/quickstart)

<Note>
  **クイックヒント:** `!`を押すとbashモードに切り替わり、AI解釈なしでシェルコマンドを直接実行できます。`Esc`を押すと通常モードに戻ります。詳細は[CLI リファレンス](/jp/reference/cli-reference#bash-mode)を参照してください。
</Note>

## `droid`があなたのワークフローにもたらすもの

- **エンドツーエンドの機能開発**: 計画から実装、テストまで - droid は透明なレビューワークフローを通じてあなたの制御を維持しながら、完全な開発ライフサイクルを処理します。
- **深いコードベース理解**: リポジトリ、ドキュメント、課題追跡システム全体にわたる組織の共有知識を活用し、時間とともに向上するコンテキストに応じたアシスタンスを提供します。
- **エンジニアリングシステム統合**: 既存のツールに直接接続 — Jira、Notion、Slack、その他多くのツールとのネイティブ統合により — 開発作業がチームのプロセスと同期した状態を保ちます。
- **本番対応の自動化**: 開発中のローカル環境やCI/CDパイプラインで同じワークフローをデプロイでき、エンタープライズセキュリティとコンプライアンスが最初から組み込まれています。

## チームがFactoryを選ぶ理由

- **エンタープライズ向けに構築**: オンプレミス デプロイメント オプション、SOC-2コンプライアンス、エアギャップ環境。私たちは最低価格競争よりもセキュリティと品質を優先します。
- **あなたのツールを強化**: 既存のターミナル、IDE、開発環境内で動作します。エディターを切り替えたり新しいインターフェースを学んだりする必要はありません — droidはあなたが既に生産性を発揮している場所で統合されます。
- **透明で制御可能**: droidが行うすべての決定は可視化され、レビュー可能です。ネイティブdiffビューアーと承認ワークフローにより、コード変更の完全な監視を維持します。
- **モデルの柔軟性**: 単一のAIプロバイダーにロックインされません。Factoryでは組織全体で一貫した動作とメモリを維持しながら、各ジョブに最適なモデルにタスクをルーティングできます。
