<Update label="3月6日" rss={{ title: "CLI アップデート", description: "Git worktreeサポート、ネイティブPDF読み取り、ランタイム設定オーバーレイ" }}>
  `v0.70.0`

  ## 新機能

  * **Git worktreeサポート** - 新しい`--worktree`フラグにより、複数のブランチで同時作業するためのネイティブgit worktreeサポートが有効化
  * **ネイティブPDFサポート** - ReadツールがすべてのモデルプロバイダーでネイティブにPDFファイルの読み取りをサポート
  * **ランタイム設定オーバーレイ** - 新しい`--settings`フラグにより、組織優先マージでランタイム時に設定を適用可能
  * **リモートセッション用Skills UI** - Skills UIがdroidコンピューターセッションで利用可能になりました（アプリ）

  ## 改善

  * **Mission Controlの再設計** - よりクリーンなレイアウトでMission Controlインターフェースをリフレッシュ

  ## バグ修正

  * **Option+Arrowによる単語ジャンプ** - ESCプレフィックスターミナルでのOption+Arrow単語ジャンプを復元
  * **バックグラウンドプロセスの分離** - バックグラウンドプロセストラッキングがCLIインスタンスごとに分離され、インスタンス間の干渉を防止
  * **@ 検索インデックス** - @ 検索候補でのファイルインデックスを修正
  * **デーモンでのシェル環境** - デーモンがシェル環境変数を正しく継承するようになりました
  * **ターミナル互換性** - 特定のターミナルで問題を引き起こす可能性があったKittyキーボードプロトコル検出を修正
  * **ストリーミングキャンセル** - ストリーミングレスポンスをキャンセルする際の信頼性を向上
  * **デーモンの自動更新** - デーモンの自動更新が誤ってスキップされる問題を修正

</Update>
