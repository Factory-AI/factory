<Update label="3月16日" rss={{ title: "CLIアップデート", description: "設定可能なLLMタイムアウト、Enterprise Controls用のMCPサーバー許可リスト、ミッションコントロールの改善" }}>

## 改善

- **ミッションコントロールショートカット** - より素早いアクセスのため、ミッションコントロールがCtrl+Tで切り替えできるようになりました

## バグ修正

- **モデル選択テキスト色** - モデル選択ドロップダウンの不正なテキスト色を修正しました（アプリ）
- **Enterpriseモデルデフォルト** - クラウドセッションがエンタープライズモデルのデフォルトを正しく尊重するようになりました（アプリ）
- **モデルポリシーオーバーライドモーダル** - モデルポリシーオーバーライドモーダルの表示問題を修正しました（アプリ）

</Update>
