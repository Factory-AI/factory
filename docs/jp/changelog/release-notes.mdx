<Update label="4月7日" rss={{ title: "CLI更新", description: "新しい/install-qaコマンド、design.mdサポート、Cmd+Nで新しいセッション、CLIの安定性修正" }}>
  `v0.96.0`

## 新機能

- **`/install-qa`コマンド** - プロジェクト用の自動QAスキルを生成する新しい`/install-qa`コマンド
- **`design.md`サポート** - CLIが`agents.md`と並んで`design.md`ファイルをサポートし、Droidにデザインガイドラインを提供
- **Cmd+Nで新しいセッション** - Cmd+Nキーボードショートカットを使用して新しいセッションをすぐに開始（アプリ）
- **セッションの曖昧さ解消** - 同じ名前のセッションが識別しやすいようにツールチップで完全パスを表示（アプリ）
- **セッション前バグレポート** - アクティブなセッションがない場合でもデスクトップアプリからバグレポートを送信（アプリ）

## 改善

- **Bashモード出力の再設計** - Bashモード出力がexecuteツール表示と一致し、より一貫性のある外観
- **MCPツール確認表示** - 読みやすさを向上させるため確認ポップアップのMCPツール表示をリスタイリング
- **ライトモードの色** - ハードコードされた色をテーマシステムに移行してライトモードの色の一貫性を向上
- **アーカイブされたセッションへのアクセス** - アーカイブされたセッションを読み取り専用モードで表示し、直接アクセス時に復元可能（アプリ）
- **一貫したホバー状態** - すべてのオプション項目で一貫したホバー状態のスタイリング（アプリ）
- **デスクトップUIの洗練** - デスクトップアプリ全体の様々な視覚的改良（アプリ）

## バグ修正

- **重複ツール呼び出しメッセージ** - 重複ツール呼び出しメッセージとツール実行中の更新の欠落を修正
- **ミッション機能カウント** - `/missions`の機能カウントが正しくない問題を修正
- **Escapeキー処理** - Escapeキーを押した後のプロンプト処理問題を修正
- **ターミナル入力セキュリティ** - セキュリティ向上のためターミナル入力キャプチャを強化
- **セッション作成** - 空の作業ディレクトリでセッションが作成される可能性がある問題を修正（アプリ）
- **プロファイルドロップダウンスタイリング** - プロファイルドロップダウンとメニュー項目のスタイリングを調整（アプリ）
- **フォントサイズの一貫性** - Computersセクションのフォントサイズの不一致を修正（アプリ）
- **サイドバーレイアウト** - セッショングループヘッダーがサイドバーの全幅にまたがるよう修正（アプリ）

</Update>
