<Update label="4月21日" rss={{ title: "CLIアップデート", description: "新しい/btwコマンド、アップグレードされた/copyセレクター、プロジェクト全体のセキュリティ監査、CLIの安定性修正" }}>
  `v0.106.0`

## 新機能

- **`/btw`コマンド** - 現在のセッションにサイドメッセージを送信するための新しい`/btw`スラッシュコマンド
- **アップグレードされた`/copy`セレクター** - `/copy`コマンドのセレクターUIを改善
- **プロジェクト全体のセキュリティ監査** - セキュリティレビュースキルに新しいプロジェクト全体監査モード
- **`/copy`スラッシュコマンド** - `/copy`スラッシュコマンドとユーザーメッセージホバーアクションがアプリで利用可能になりました（アプリ）
- **セッションIDで検索** - セッションIDで直接セッションを検索（アプリ）
- **メンバーロールフィルター** - 組織メンバーリストをロールでフィルタリング（アプリ）

## 改善

- **タイムアウトエラーハンドリング** - より長い再試行遅延でタイムアウトエラーの処理を改善
- **企業ネットワークエラー検出** - 企業のファイアウォール、DNS、TLSエラーは再試行不可能として扱われるようになりました
- **コンテキスト制限の分類** - モデルのコンテキストウィンドウを超えるリクエストは、より明確なメッセージでコンテキスト制限エラーとして分類されます

## バグ修正

- **サブエージェント権限リクエスト** - サブエージェント権限リクエストはセッションをクラッシュさせる代わりに自動拒否されるようになりました
- **コマンド後の自動提案** - コマンドが受け入れられた後に自動提案が表示されなくなりました
- **認証エラーの再試行** - 401および403認証エラーは再試行されなくなりました
- **デフォルトモデルの保持** - ユーザーが設定したデフォルトモデルは読み取り時に書き換えられなくなりました
- **却下されたハンドオフアイテム** - 却下されたハンドオフアイテムは検証中に保持されるようになりました
- **バグレポートフォールバック** - バックグラウンドデーモンが利用できない場合、バグレポートはローカル送信にフォールバックするようになりました
- **Factory APIキーのスコープ** - Factory APIキーリストは現在の組織にスコープされるようになりました（アプリ）
- **ページ間でのSlackセッションオーナー** - 保存されたSlackセッションオーナーが、ページネーション化された組織メンバーリスト全体で正しく表示されるようになりました（アプリ）
- **単一のピン留めセッション** - サイドバーで複数のピン留めセッションを許可する問題を修正（アプリ）

</Update>
