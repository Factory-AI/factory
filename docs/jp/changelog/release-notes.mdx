## 改善点

- **セッション再読み込みの高速化** - セッションの再読み込みが大幅に高速化され、よりスムーズな体験を提供します

- **Todoリスト表示** - todoリスト表示の視覚的な問題を修正しました
- **カスタムモデル検証** - ミッションワーカーでカスタムモデルを使用する際の検証エラーを修正しました
- **Mission Controlモデルセレクター** - Mission Controlのモデルセレクターを修正しました

- **`droid exec`ストリーミングJSON入力モード**: より良い自動化ワークフローのため、マルチターンexecセッション用のストリーミングJSON入力モードを追加
- **`droid exec`ツール設定**: droid execセッションでの特定ツールの有効化/無効化機能を追加
- **`droid exec`事前作成セッションID**: より良いセッション管理のため、droid execで事前作成セッションIDのサポートを追加
- **初期プロンプトサポート**: コマンドライン経由で初期プロンプトとともにTUIを起動するサポートを追加
- **GLM-4.6サポート**: Fireworks、Baseten、DeepInfraプロバイダーでGLM-4.6のLLMプロキシサポートを追加
- **GPT-5用Azure OpenAI**: TUIでGPT-5 Codex用Azure OpenAIサポートを追加
- **カスタムドロイド生成**: より高速なセットアップのため、LLMを使用したカスタムドロイドの自動生成を追加
- **MCPストリーマブルHTTPサーバー**: ストリーマブルHTTP MCPサーバーのサポートを追加
- **モデルセレクターリフレッシュ**: 改善された整理と画像サポート警告でモデルセレクターUIを更新
- **Tabキー自動補完**: より良いUXのため、Tabキーが送信せずにカスタムコマンドを自動補完
