<Update label="4月10日" rss={{ title: "CLIアップデート", description: "段階的wikiリフレッシュ、テーマ化されたmermaid図表、Deep Wiki検索、CLIの安定性修正" }}>
  `v0.99.0`

## 新機能

- **段階的wikiリフレッシュ** - Wikiリフレッシュは変更されたページのみを更新し、より高速な更新を実現
- **テーマ化されたmermaid図表の色** - Mermaid図表は現在のテーマに合わせた要素ごとの色でレンダリング
- **Deep Wiki検索とエクスポート** - Wikiでキーワード検索、スキルブラウジング、ページエクスポートをサポート（アプリ）

## バグ修正

- **テーマ背景オーバーライド** - 予期しないターミナルテーマ背景のオーバーライドを修正
- **BYOK ベースURL切り替え** - カスタムAPIベースURL切り替え後にクライアント接続が更新されない問題を修正
- **コンテキストモデル継承** - コンテキスト管理が現在のセッションモデルを使用していない問題を修正
- **仕様エディターターミナル引き渡し** - 仕様エディターがターミナル引き渡しを適切に完了しない問題を修正
- **MCP認証キャンセル** - MCPパネルを閉じる際に保留中のMCP認証がキャンセルされない問題を修正
- **ツール結果表示** - ツール結果に非文字列値が含まれる際のクラッシュを修正
- **カスタムモデルの安定性** - セッション間でカスタムモデルIDが予期せず変更される問題を修正
- **ディレクトリが見つからないエラー** - ディレクトリが見つからないエラーがディレクトリ確認時に即座に表示されるように修正（アプリ）
- **BYOK使用制限** - BYOKユーザーがプラットフォーム使用制限に達してもチャットを継続可能に修正（アプリ）
- **ゴーストツールコール** - メッセージ処理中に幻のツールコールが表示される問題を修正
- **チャット入力** - 様々なチャット入力の問題を修正（アプリ）
- **コンピューターターミナル** - Droid Computers使用時のターミナル表示問題を修正（アプリ）
- **マシン設定モーダル** - マシン設定モーダルが正しく表示されない問題を修正（アプリ）
- **推論とモデル選択** - 推論レベルとモデル選択の問題を修正（アプリ）

</Update>
