- **承認での完全実行コマンド** - 承認ダイアログで実行される完全なコマンドが表示されるようになりました

- **マシン変更時のローカル再接続** - 接続を切り替える際のローカルマシン再接続が確実に動作するようになり、接続されていない場合の警告がより明確になりました（アプリ）
- **ツールレンダリングの整合性** - DefaultToolとTaskToolのレンダリングがCLIと一致するよう改善（アプリ）
- **ApplyPatchの差分レンダリング** - ApplyPatchツールでCLI入力フォーマット用の差分レンダリングを修正（アプリ）

</Update>
