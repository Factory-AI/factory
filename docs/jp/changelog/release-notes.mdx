<Update label="4月3日" rss={{ title: "CLI更新", description: "/limits使用パネル、クレジット使用カウントダウン、モデルセレクター拡張、CLI安定性修正" }}>
  `v0.94.0`

## 新機能

- **/limits使用パネル** - 新しい`/limits`コマンドで、バケット表示付きのインタラクティブなクレジット使用量の内訳を表示
- **クレジット使用カウントダウン** - 残りのクレジット制限をリアルタイムで表示するライブカウントダウンタイマー（アプリ）
- **モデルセレクター拡張** - モデルセレクターに「もっと見る」オプションを追加し、追加モデルを閲覧可能（アプリ）

## 改善

- **キューメッセージの再設計** - キューに入ったメッセージが、よりクリーンなユーザーメッセージスタイルのレイアウトで表示されるように
- **/pluginsと/skillsメニューの刷新** - `/plugins`と`/skills`メニューが、更新された`/sessions`と`/model`のUIスタイルに合致するように
- **Droid Shieldプレースホルダー検出** - Droid Shieldが追加のプレースホルダー値パターンを検出し、偶発的なシークレット露出をより適切に防止

## バグ修正

- **ワーカーハング修正** - ブロックされたワーカーがCLIをハングさせる可能性がある回帰を修正
- **カスタムモデルストリーミング** - カスタムモデル使用時のストリーミングエラーを修正
- **長いプロンプト処理** - サブエージェントに渡される際の長いプロンプトが失敗する問題を修正
- **自律レベルサイクリング** - Ctrl+Lが自律レベルを正しくサイクルしない問題を修正
- **Kittyターミナル修正** - Kittyターミナルでの追加の入力処理問題を修正

</Update>
