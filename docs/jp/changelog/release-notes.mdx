  `v0.92.0`

## 新機能

- **統合 /missions メニュー** - モデルタブと改善されたナビゲーションを備えた `/missions` コマンドの再設計
- **/settings の再設計** - 更新されたレイアウトによる `/settings` メニューのリフレッシュ
- **CLI フッターの再設計** - よりクリーンなレイアウトでCLIフッターを再設計
- **ワーカー別トークン詳細** - Mission Control でワーカー別のトークン使用量詳細を表示
- **セッションフォーク** - 既存のセッションをフォークして新しいセッションを作成する `--fork` フラグを追加
- **TUI チャートレンダリング** - チャートとグラフがターミナルで直接レンダリング
- **Exec モードシステムプロンプトフラグ** - Exec モードでシステムプロンプトをカスタマイズするための `--append-system-prompt` および `--append-system-prompt-file` フラグを新規追加
- **クラウド同期読み取り専用セッション** - クラウド同期セッションが読み取り専用モードで表示可能（アプリ）
- **モデル別コンテキスト制限設定** - モデル別にコンテキストトークン制限を設定する新しい設定

## 改善点

- **セッション読み込みパフォーマンス** - より高速な起動と短縮された読み込み時間のためのセッション読み込み最適化
- **より高速なセッション検索** - セッション検索が大幅に高速化（アプリ）
- **より高速なデスクトップセッション読み込み** - デスクトップアプリでセッションがディスクから直接読み込まれ、速度が向上（アプリ）
- **ツールルーティング警告** - ツールルーティング警告でモデル固有の代替案を提案
- **デフォルトモデル表示** - `/model` コマンドでデフォルトモデルがハイライト表示
- **SKILL.md サポート** - 組み込みスキルを SKILL.md ファイルから読み込み可能
- **サイドバーの洗練** - サイドバーのユーザープロフィールとドロップダウンアニメーションをモダン化（アプリ）

## バグ修正

- **SSH 経由のクリップボードコピー** - SSH 接続でクリップボードコピーが確実に動作
- **VS Code マルチライン入力** - VS Code ターミナルでのマルチライン入力処理を修正
- **MCP 認証** - MCP OAuth 認証フローを復旧
- **PDF ページ制限エラー** - 100ページのPDF制限を超過した際の明確なエラーメッセージを追加
- **終了時のターミナルクリーンアップ** - CLI 終了後にターミナルエスケープシーケンスが残る問題を修正
- **スラッシュコマンド設定** - `/fast`、`/enter-mission`、`/exit-mission`、`/new` コマンドが設定を正しく適用しない問題を修正
- **並列ツールレンダリング** - 並列ツール実行でのレンダリング問題を修正
- **Ghostty と xterm の入力** - Ghostty と xterm ターミナルでのキーボード入力処理を修正
- **ターミナル多重化表示** - tmux や screen などのターミナル多重化ツールでの画面クリアを修正
- **ターミナルフォーカス復旧** - ウィンドウ切り替え後にターミナルフォーカスが復旧しない問題を修正
- **AGENTS.md セッション読み込み** - 既存セッション再開時に AGENTS.md ガイドラインが正しく読み込まれるよう修正
- **Task ツールバックグラウンドパネル** - Task ツールバックグラウンドパネルでのストリーミングと表示問題を修正
- **フルスクリーンサイドバーレイアウト** - macOS でのフルスクリーンモードでのサイドバーレイアウトを修正（アプリ）
