<Update label="4月2日" rss={{ title: "CLI アップデート", description: "GPT-5.3-Codex fast mode、マークダウンリンク、agents.md自動検出、セッション保持ポリシー" }}>
  `v0.93.0`

## 新機能

- **GPT-5.3-Codex fast mode** - より高速なコード生成のため、GPT-5.3-Codexをfast modeモデルオプションとして追加
- **マークダウンリンクレンダリング** - マークダウンリンクがCLIでクリック可能なリンクとしてレンダリング
- **agents.md自動検出** - CLIがプロジェクト内のagents.mdファイルを自動検出して読み込み
- **セッション保持ポリシー** - ストレージ管理のためのセッション自動削除ポリシーを設定 (app)

## 改善

- **プラン更新インジケーター** - プランが変更されたときに控えめな「プラン更新済み」通知を表示
- **チャット内JSON レンダリング** - JSONデータがウェブとデスクトップチャットで改良されたフォーマットでレンダリング (app)
- **自律性リクエスト付近のコマンド** - 自律性リクエストの近くに提案されたコマンドを表示し、簡単にアクセス可能に (app)

## バグ修正

- **レビューでLGTMを表示** - reviewコマンドが問題を見つけなかった場合に「LGTM」を表示
- **長い出力の切り詰め** - Executeツールプレビューの長い出力行がターミナル幅に合わせて切り詰め
- **メッセージのゴースト化** - キューに入れられたメッセージを送信時にゴーストメッセージが表示される問題を修正
- **エラーメッセージの改善** - モデルストリーミングエラーが生のエラーではなくユーザーフレンドリーなメッセージを表示
- **セッション読み込み** - セッションの読み込みと再開に関する各種問題を修正
- **オンボーディングリンク** - オンボーディングフローの壊れたリンクを修正 (app)
- **ミッションモード入力** - ミッションモードでの入力フィールドの動作を修正 (app)

</Update>
