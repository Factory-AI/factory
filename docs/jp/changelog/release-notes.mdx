- **Kimi K2.6サポート** - Kimi K2.6モデルのサポートを追加

- **ミッションアクセスポリシー** - エンタープライズ管理者がミッションへのアクセスを制御可能

## 改善

- **BYOKのCompactionモデル** - Compactionがデフォルトで現在のモデルを使用し、BYOKに使用される

## バグ修正

- **仕様オプション選択** - ユーザー選択の仕様オプションがモデルに渡されない問題を修正
- **Executeツールヘッダー** - Executeツールヘッダーで最後の文字が切れる問題を修正
- **無効なJSONレンダリング** - JSONレンダラーで無効なJSON時に出力を隠すように修正
- **レビューガイドラインスキル** - CLIでレビューガイドラインスキルが不足時の適切な処理
- **ツールキャンセル割り込み** - フォローアッププロンプトでツールキャンセル割り込みの完了を待機
- **TUIデーモンセッション切り替え** - 前のTUIデーモンセッションは次のセッション読み込み後に終了
- **セッションタイトル更新** - デーモンセッションタイトル更新がターミナルタブに反映
- **ミッションワーカー権限** - ミッションワーカー権限リクエストが自動拒否される
- **Webチャットの思考トレース** - 思考トレースがWebセッションチャットに表示される
- **ripgrep起動** - バンドルされたripgrepバイナリ初期化時の断続的エラーを修正
- **切り詰められたストリーム処理** - モデルレスポンスストリームが切り詰められた際のエラー処理を改善
- **固定された「さらに表示」** - 新しいコンピューターでセッション作成後の固定「さらに表示」ボタンを修正（app）
- **Slackセッション所有者選択** - 選択器でサーバーサイド検索と無限スクロールを使用（app）
- **サイドバーのホバーと選択** - サイドバーのホバー状態とハイライトを修正（app）
