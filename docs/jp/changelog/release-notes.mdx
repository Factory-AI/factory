## 新機能

- **コードレビュースキルのインストール** - プロジェクトで自動化されたコードレビューを設定するための新しい組み込みスキル

- **macOS Spotlight発見性** - Factory DesktopがmacOS Spotlight検索で見つけやすくなりました (app)

## 改善

- **QAスキルインストールアップグレード** - install-qaスキルがテスト失敗から学習して時間の経過とともに改善されるようになりました
- **wiki読み込み高速化** - wikiページの読み込みが最適化され、遅延が削減されました (app)

## バグ修正

- **ターミナルパレット上書き** - ターミナルのカラーパレットが予期せず上書きされる問題を修正
- **Droid Shield SSH鍵検出** - Droid ShieldがSSH公開鍵を誤ってシークレットとしてフラグする問題を修正
- **Droid Computerセッション表示** - ローカルディレクトリなしでDroid Computersで実行された際にセッションが表示されない問題を修正 (app)
- **Droid Computer SSH** - Droid ComputersでのSSH接続問題を修正
- **ツールステータステキスト** - ツール実行中の「invoking tools」ステータスインジケーターを復元
- **仕様引き継ぎ後のEscapeキー** - 仕様承認引き継ぎ後にEscapeキーによるキャンセルが機能しない問題を修正
- **セッションサイドバー** - さまざまなサイドバー表示問題を修正 (app)
- **証明書読み込み** - 接続に影響する可能性のある証明書読み込み問題を修正
- **チャットとツール結果** - チャットメッセージとツール結果の表示問題を修正 (app)
- **ローカルマシン接続切り替え** - オフにしたときにローカルマシン接続切り替えが尊重されない問題を修正 (app)

</Update>
