Factoryは、あなたがどこで作業していても機能するAIネイティブソフトウェア開発プラットフォームです。IDEからCI/CDまで - ツール、モデル、ワークフローを変更することなく、リファクタリング、インシデント対応、マイグレーションなどの完全なタスクをDroidに委任できます。

  className="rounded-lg"
/>

## なぜ私たちを選ぶのか？

* **モデル非依存：** OpenAI、Anthropic、Google、X AI、オープンソース、またはローカルモデルを使用。
* **コンテキスト信頼性：** アンカーポイントを使用して段階的に圧縮し、ほぼ完璧なコンテキストを実現。
* **仕様モード：** 詳細な実装、技術的詳細、混合モデルサポート。
* **エンタープライズ対応：** SSO + SAML、専用コンピュート、コンプライアンス＆セキュリティなど。
