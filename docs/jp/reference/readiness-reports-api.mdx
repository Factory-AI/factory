Readiness Reports APIを使用すると、アクセス権のあるリポジトリのエージェント準備状況レポートを取得できます。

---

## 認証

すべてのリクエストで、`Authorization`ヘッダーにFactory APIキーが必要です。APIキーは[app.factory.ai/settings/api-keys](https://app.factory.ai/settings/api-keys)で生成できます。

```bash
Authorization: Bearer fk-your-api-key
```

## ベースURL

```text
https://app.factory.ai
```

## Readiness reportの一覧取得

組織のreadiness reportを、あなたがアクセスできるレポートに絞って取得します。

```http
GET /api/organization/agent-readiness-reports
```

レポートは、接続済みのSCMアカウントが対象リポジトリにアクセスできる場合、またはレポートを生成したDroidセッションをあなたが作成した場合に返されます。

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `repoId` | string | いいえ | リポジトリIDでレポートを絞り込みます |
| `limit` | integer | いいえ | 返すレポート数の上限。無効な値はサービス既定値になります。 |
| `startAfter` | string | いいえ | ページネーション用のレポートIDカーソル |

<Note>
  古い連携では`/api/organization/maturity-level-reports`を参照している場合があります。新しい連携では`/api/organization/agent-readiness-reports`を使用してください。
</Note>
